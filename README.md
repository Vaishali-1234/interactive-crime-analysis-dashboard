# Interactive Crime Analysis Dashboard

This project focuses on analyzing crime data to identify trends, hotspots, and patterns using interactive visualizations and data analytics techniques.

## 🚀 Features
- Crime Hotspot Detection using Heatmaps
- Crime Type Distribution Analysis
- Time-based Crime Trend Analysis
- Location-wise Crime Statistics
- KPI Metrics such as Arrest Rate and Unsolved Cases
- Interactive Filtering and Dashboard Visualization

## 📊 Tools & Technologies
Python | Pandas | NumPy | Matplotlib | Seaborn | Plotly | Jupyter Notebook

## 📁 Dataset
Public crime dataset containing features such as crime type, location, date, arrest status, and response time.

## 🎯 Outcome
The dashboard helps in understanding crime patterns and supports data-driven decision-making for safety and law enforcement planning.
